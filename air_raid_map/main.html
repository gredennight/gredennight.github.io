Панель керування Мапою Тривог

1) Оновлення ПЗ контролера

Оновіть контролер мапи до останньої версії програмного забезпечення. Для цього спочатку підключіть мапу кабелем для передачі даних до комп'ютера. На зворотній стороні мапи знайдіть контролер, затисніть малу кнопку яка знаходиться поряд з червоним світлодіодом. На сторінці натисність Прошити контролер мапи, оберіть порт, натисніть Install AirRaidMap, натисність Install та дочекайтеся завершення процесу. Після цього можете відпустити кнопку на контролері.
0%

2) Під'єднання до мапи

Для налаштувань мапи необхідно підключитися до неї. Для цього з'єднайте мапу кабелем для передачі даних з комп'ютером та натисніть Підключитися до контролера, оберіть необхідний порт.

3) Налаштування Wi-Fi для отримання даних

Введіть дані 2.4ГГц мережі Wi-Fi, через яку мапа буде оновлювати дані про стан областей

4) Налаштування областей

Налаштуйте яскравість та кольори для кожного стану тривоги

5) Керування

Натисніть надіслати налаштування для оновлення параметрів мапи, після цього натисніть перезапустити мапу щоб їх застосувати
